类的方法分为三大类：
1.构造方法（无参构造、有参构造）,
当有多个有参构造时，参数列表不同（类型或个数），表示方法重载
例如：
public class Dog{
	private String name;//当用this.name时，调用该属性
	//默认隐式存在无参构造方法(当手动编写有参构造时，该功能失效)
	public Dog(){}
	//有参构造可以有多个，只要参数列表中类型或个数不同即可
	public Dog(String name){
		this.name = name;
	}
}

..main(){
	Dog dog1 = new Dog();
	Dog dog2 = new Dog("二狗");
}

2.静态方法（例如：main主程序入口方法）
3.实例方法（通过对象名“.”出来的方法）
